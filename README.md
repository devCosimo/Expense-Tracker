## Installation

1. Clone the repository
```bash
git clone https://github.com/devCosimo/expense-tracker.git
```

2. Install dependencies
```bash
cd expense-tracker
npm install
```

3. Run the development server
```bash
npm run dev
```

## Dependencies

- React
- Recharts
- Lucide React
- Tailwind CSS
